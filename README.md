# Promedio Semanal del Clima
Este proyecto contiene dos soluciones para calcular el promedio semanal de temperaturas:

1. **Programación Tradicional:**  
   Utiliza funciones para ingresar las temperaturas y calcular el promedio.

2. **Programación Orientada a Objetos (POO):**  
   Utiliza una clase para organizar y calcular las temperaturas.

# Instrucciones:
1. Clonar el repositorio.
2. Ejecuta los scripts `python promedio_tradicional.py` o `python promedio_poo.py` según el método que prefiera probar.
3. Ingresar las temperaturas diarias de la semana y se obtendrá el promedio.

En el archivo `comparacion.md`, se observa una comparación de ambos enfoques.
